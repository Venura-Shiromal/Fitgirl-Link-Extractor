<h1>🎮 FitGirl Link Extractor</h1>

</p>

<p>A fast and automated Python script to extract <code>fuckingfast</code> download links directly from FitGirl Repacks game pages. Skip the manual clicking and get your links instantly saved to a text file.</p>

<h2>✨ Features</h2>
<ul>
  <li>🚀 <strong>Automated Extraction:</strong> Instantly grabs all <code>fuckingfast</code> links from a given game page URL.</li>
  <li>📁 <strong>Clean Output:</strong> Automatically saves all extracted links into a neatly formatted <code>output.txt</code> file.</li>
  <li>⚡ <strong>Lightweight:</strong> Built with minimal dependencies for quick execution.</li>
</ul>

<h2>🛠️ Prerequisites</h2>
<p>Ensure you have <a href="https://www.python.org/downloads/">Python 3.8 or higher</a> installed on your system.</p>

<h2>🚀 How to Use</h2>

<p><strong>1. Install requirements</strong><br>
Open your terminal in the project directory and install the necessary dependencies:</p>
<pre><code class="language-bash">pip install -r requirements.txt</code></pre>

<p><strong>2. Run the script</strong><br>
Execute the Python file using your terminal:</p>
<pre><code class="language-bash">python fitgirl_link_extractor.py</code></pre>
<p><em>(Note: Depending on your system, you can also use <code>py fitgirl_link_extractor.py</code>)</em></p>

<p><strong>3. Paste the link</strong><br>
When prompted by the program, paste the full URL of the game from the FitGirl website and press Enter.</p>

<p><strong>4. Get your links</strong><br>
The script will run and the extracted links are automatically written into an <code>output.txt</code> file in the same folder. You're done!</p>
