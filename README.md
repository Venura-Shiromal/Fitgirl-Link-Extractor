# 🎮 FitGirl Link Extractor

![Python](https://img.shields.io/badge/python-3.6+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

A fast and automated Python script to extract `fuckingfast` download links directly from FitGirl Repacks game pages. Skip the manual clicking and get your links instantly saved to a text file.

## ✨ Features
- 🚀 **Automated Extraction:** Instantly grabs all `fuckingfast` links from a given game page URL.
- 📁 **Clean Output:** Automatically saves all extracted links into a neatly formatted `output.txt` file.
- ⚡ **Lightweight:** Built with minimal dependencies for quick execution.

## 🛠️ Prerequisites
Ensure you have [Python 3.x](https://www.python.org/downloads/) installed on your system.
